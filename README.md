# IT-Operations-Analytics-Dashboard
* Processed and analyzed 97,498+ IT support records to uncover operational trends and service bottlenecks.
* Built an interactive dashboard with 10+ KPIs, improving reporting efficiency and real-time decision-making.
* Identified 72,382 overdue tickets and monitored 4.55 average resolution time to enhance SLA compliance.
* Tracked a 4.10 customer satisfaction (CSAT) score to evaluate support quality and customer experience.
* Used Pivot Tables, XLOOKUP, Slicers, KPI Cards, and dynamic charts to automate analysis and improve productivity.
